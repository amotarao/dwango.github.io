  •  VRM(ja)

    • VRM - 3D avatar file format for VR -

    “VRM” is a file format for handling humanoid 3D avatar (3D model) data for VR applications. It is based on glTF 2.0. Anyone can use it for free. Here we provide an open source software (UniVRM) implemented in C# that can read and write VRM file in Unity.
